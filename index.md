# C++ A* Algorithm Project - Rupert Claisse

## Structure

My project consists of three files: main.cpp, AStar.h, and AStar.cpp. Inside the header file, the AStar class contains a public getter and a private 2D vector grid, the latter being created with ChatGPT.

In main.cpp, an instance of the AStar class is created and std::cout is used to output the values in the grid.
